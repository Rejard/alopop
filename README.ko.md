# Alopop - 팩트체크 AI 메신저
[🇺🇸 Read in English (영어로 읽기)](./README.md)

![Alopop Architecture](./public/alopop-architecture.svg)

**Alopop**은 단순한 실시간 채팅 애플리케이션을 넘어선 혁신적인 **"팩트체크 AI 메신저"**입니다. 일상적인 대화 속에서 자연스럽게 가짜 뉴스, 조작된 미디어, 딥페이크를 검증하여 모든 데이터 패킷의 진실성을 판별하도록 설계되었습니다.

---

## 💡 핵심 철학과 엔지니어링 비전

> **"채팅창을 흐르는 모든 메시지, 이미지, 데이터의 절대적 진실을 검증한다."**

초연결 사회에서 메신저는 거짓 정보가 퍼지는 가장 빠른 통로가 되었습니다. 이러한 문제를 시스템 차원에서 해결하기 위해, Alopop은 유저 간의 대화 맥락을 이해하고 DuckDuckGo 등을 통한 실시간 인터넷 검색을 수행하여 정보의 진위를 즉각적으로 판별하는 혁신적인 코어 엔진으로 구동됩니다.

### 왜 Alopop인가? (주요 차별점)
- **실시간 교차 검증 (Real-Time Cross-Validation)**: 채팅 중 의심스러운 주장(예: 논란이 되는 뉴스, 왜곡된 사실)이나 딥페이크가 감지되면, 코어 AI 엔진이 병렬 웹 스크래핑과 비전 분석을 실행하여 진실을 검증합니다. 이후 채팅 말풍선에 **팩트체크 배지**를 자동으로 부착합니다.
  
  ![Fact Check UI](./public/factcheck-ui.svg)
  
  - **🤖 AI 생성 이미지 배지**: 전송된 사진이 AI로 조작되었거나 합성된 딥페이크일 확률을 시각화합니다.
  - **🚨 가짜 뉴스 경고 배지**: 피싱 시도, 가짜 뉴스, 사실 왜곡이 감지되면 빨간색 경고 배지를 부착하여 오보의 확산을 방지합니다.
  - **🔍 실시간 근거 팝업**: 배지를 클릭하면 실시간 교차 검증을 통해 획득한 **판별 근거(Evidence)**와 **AI 신뢰도 점수(%)**가 담긴 팝업이 표시되어 유저가 올바른 판단을 내릴 수 있도록 돕습니다.
- **제로 트러스트 노-로그 아키텍처 (Zero-Trust No-Log)**: 중앙 집중식 채팅 저장소 없이 소켓 릴레이 방식으로만 작동하여 완벽한 개인정보 보호와 데이터 주권을 보장합니다.

## 🚀 엔터프라이즈급 핵심 기능

Alopop은 팩트체크 코어 엔진 위에 구축된 강력하고 커스터마이징 가능한 생태계를 제공합니다.

### 1. 멀티 도메인 개인 AI 스튜디오 & 오케스트레이션 🛠️
Alopop은 단순한 챗봇 인터페이스를 넘어, 모든 유저에게 **멀티 도메인 개인 AI 스튜디오**를 제공합니다.
- **독립된 전용 워크스페이스**: 유저가 접속하면 게임 개발, 공연 기획, 법무법인, 세무회계법인 등 4개의 독립된 전문 스튜디오가 자동으로 부여됩니다.
- **다중 에이전트 파이프라인 (Multi-Agent)**: 각 스튜디오는 기획자, 디자이너, 개발자, QA 등 직군별로 세분화된 에이전트들로 구성됩니다. 이들은 실시간 웹소켓 통신을 통해 유저의 복잡한 프로젝트 요구사항을 자율적이고 협력적으로 완수해 나갑니다.
- **자가 치유 아키텍처 (Auto-Healing)**: 스튜디오 템플릿 엔진(`studio_templates.json`)을 통해 에이전트 설정이 훼손되거나 누락된 경우 즉각적으로 자동 복구(마이그레이션)하여 다운타임을 원천 차단합니다.

### 2. 스튜디오별 동적 AI 모델 선택기 (Advanced BYOK) 🧠
왜 하나의 AI 모델만 사용해야 할까요? Alopop의 `AiModelSelector`는 AI 인프라에 대한 전례 없는 세밀한 제어권을 제공합니다.
- **스튜디오별 모듈화**: 유저는 본인의 각 개인 스튜디오마다 전혀 다른 AI 모델을 할당할 수 있습니다 (예: 법무법인에는 OpenAI `gpt-4o`, 게임 개발에는 Anthropic `claude-3.5-sonnet`, 기본 팩트체크에는 Google `gemini-1.5-pro`).
- **독립된 키 관리**: BYOK(Bring Your Own Key) 아키텍처가 동적으로 작동하여, 모델 제공자(Provider)에 맞는 올바른 API 키를 실시간으로 매핑하여 비용 효율성과 모델 전문성을 극대화합니다.

### 3. Pet365Care 통합 에코시스템 🐾
과거 독립된 서비스로 운영되던 **Pet365Care** (반려동물 건강 및 소셜 관리 플랫폼)가 Alopop Next.js 에코시스템 내부에 완벽하게 통합되었습니다.
- 실시간 반려동물 건강 추적, 병원 정보 동기화, 반려동물 보호자들을 위한 소셜 피드 기능 등 모든 기능이 Alopop의 코어 인프라 위에서 구동됩니다.

### 4. 서버사이드 자율 스폰서 AI 이코노미 (BYOK)
무거운 추론(Inference) 비용을 플랫폼에 지불할 필요가 없습니다. Alopop은 LLM 인프라 주권과 수익 창출의 기회를 유저에게 돌려줍니다.
- **24/7 자율 AI 수익 창출 (Pay-Per-Use)**: 채팅방 호스트가 본인의 AI 페르소나와 API 키를 DB에 등록하면, 호스트가 앱을 켜두지 않아도 **Node.js 서버가 백그라운드에서 24시간 게스트들의 팩트체크 요청을 처리하고, 게스트의 인앱 코인을 차감하여 호스트의 지갑으로 자동 입금**합니다. 웹3 스타일의 거대한 패시브 인컴 패러다임입니다.
- **군사급 DB 암호화**: 유저가 제출한 외부 API 키는 절대 평문으로 저장되지 않습니다. 초기화 벡터가 적용된 **AES-256-CBC 암호화 알고리즘**으로 강력하게 보호되어, 마스터 DB가 해킹되더라도 고한도 상업용 키를 안전하게 지켜냅니다.

### 5. 초개인화 AI 페르소나 (MBTI 엔진)
일반적인 챗봇을 넘어, Alopop은 채팅방에 적극적으로 참여하는 고도로 개인화된 AI 동반자를 설계할 수 있게 해줍니다.
- **정밀 MBTI 주입**: AI 파트너의 **16가지 MBTI 성격 유형, 나이, 성별, 직업, 구체적인 말투(예: 비꼬는 말투, 공감하는 말투)**를 시스템 프롬프트를 통해 커스터마이징 할 수 있습니다.
- 이 맞춤형 AI 친구는 채팅방에 상주하며 대화의 맥락에 완벽하게 녹아들어 팩트체크를 수행합니다.

### 6. P2P 스마트 코인 이코노미 & 실시간 송금
채팅방 호스트가 API 추론 비용을 전부 부담할 필요가 없습니다. 내부 가상 코인 원장을 활용하여 채팅방 내부에 세밀한 비용 분담 정책을 도입했습니다.
- **수수료 제로 P2P 송금**: 가상 인앱 '코인' 지갑 시스템이 통합되어 있어, 웹소켓을 통해 수수료 없이 참가자들끼리 팁을 주거나, 정보에 대한 대가를 지불하거나, AI 요청 자금을 조달할 수 있습니다.
- **전례 없는 과금 유연성**: 
   1) **개별 라우팅**: 모든 사람이 각자의 API 키를 사용합니다.
   2) **스폰서 모드 (Pay-Per-Use)**: 게스트가 호스트의 AI 엔진을 사용하기 위해 호스트가 정한 '코인 톨게이트 비용'을 자동으로 지불합니다.
   3) **P2P 코인 풀링**: 무거운 멀티모달리티 요청을 처리하기 위해 참가자들이 코인을 모금(더치페이)하여 공동으로 자금을 조달합니다.

### 7. 관리자 지원 "무료 AI 이벤트" & CMS 대시보드 👑
API 키가 없는 신규 유저의 진입 장벽을 낮추기 위해, 관리자에게 강력한 **CMS (콘텐츠 관리 시스템) 대시보드**를 제공합니다.
- **마찰 없는 온보딩 (무료 AI 이벤트)**: 관리자는 동적으로 특정 모델(예: Gemini 1.5 Pro)에 대한 "무료 AI 이벤트"를 발행할 수 있습니다. 시스템은 BYOK 설정이 없는 신규 유저를 자동 감지하여 관리자의 금고에서 지불되는 무료 AI 추론 일일 할당량을 부여하고, 보라색 `[EVENT]` 뱃지를 표시합니다. 할당량이 소진되면 깔끔하게 기본 BYOK 요구 상태로 전환됩니다.

### 8. 로컬 우선 아키텍처 (Local-First) & 낙관적 UI ⚡
- **IndexedDB (Dexie.js) 캐싱 엔진**을 내장하여, 메시지가 브라우저 로컬 데이터베이스에 선제적으로 영구 저장되어 즉각적인 낙관적 UI(Optimistic UI) 업데이트를 보장합니다. 네트워크 상태가 매우 안 좋을 때도 지연 없는 스크롤링 경험을 제공합니다.

### 9. 매끄러운 PWA & 네이티브 오프라인 푸시 (Web Push) 📲
- **오프라인 메시지 큐**: 네트워크가 끊긴 상태에서 보낸 메시지는 안전하게 캐시되었다가 연결 시 백그라운드 큐를 통해 자동 동기화됩니다.
- **VAPID 웹 푸시**: 서비스 워커(Service Worker)를 통해 유저 기기에 OS 수준의 네이티브 푸시 알림을 즉각적으로 전송하여, 앱스토어 설치의 마찰 없이도 앱과 같은 사용자 유지를 달성합니다.

### 10. 하이브리드 소프트 온보딩 & 21개 미니게임 포털 🎮
- **게스트 접속**: 가입 장벽을 완전히 허물었습니다. 처음 접속 시 "게스트(익명)"로 입장하여 채팅 및 코인 지갑을 포함한 모든 기능을 자유롭게 경험해 볼 수 있습니다. 언제든 원할 때 하이브리드 계정 연동을 통해 정식 유저로 전환할 수 있습니다.
- **게임 포털**: 단순한 메시징을 넘어, 채팅 중 친구들과 함께 즐길 수 있는 **21개의 고품질 HTML5 미니게임** 포털이 내장되어 있습니다. Next.js 프록시 아키텍처를 통해 CORS 오류 없이 완벽하게 생태계에 융합됩니다.

### 11. 고스트 델리게이트 (Ghost Delegate): 하이브리드 P2P 엣지 & 서버리스 아키텍처 ♾️
- **제로 핑 섀도 리더 선출**: 수천 명의 피어가 통신 핑(ping)을 단 한 번도 발생시키지 않고, 활성화된 웹소켓 UUID를 알파벳 순으로 정렬하는 것만으로 1밀리초 만에 만장일치로 "고스트 델리게이트(임시 뇌)"를 선출합니다.
- **BYOK P2P 터널링**: '게스트 A'의 스마트폰이 무거운 AI 맥락을 계산(Edge Orchestration)하지만, 페이로드는 오프라인 상태인 '호스트 B'의 API 금고를 안전하게 복호화하도록 라우팅됩니다. 연산은 분산되지만 과금 주권은 유지됩니다.

### 12. Redis가 필요 없는 1만 CCU 인프라 & 카오스 몽키 💥
- **인메모리 속도 제한 (Rate Limiting)**: 외부 의존성 없이 작동하는 가벼운 커스텀 속도 제한기(초당 3회 요청)가 악의적인 봇 홍수나 DDOS 시도로부터 값비싼 AI API를 보호합니다.
- **통합 카오스 몽키**: 관리자 대시보드에는 '카오스 몽키' 훈련장이 내장되어 있습니다. 관리자는 수백 개의 자동화된 "고스트 요원"을 투입하여 소켓과 API에 무자비한 스팸 테스트를 가함으로써 인프라 안정성을 검증할 수 있습니다.

### 13. 완전 자율 AI 에이전트 연동 (OpenClaw 브릿지) 💻
내장된 **OpenClaw AI Agent** 브릿지를 통해 유저의 로컬 PC에 직접 연결하여, 코딩 작업, 브라우저 자동화, 시스템 운영 등을 채팅 인터페이스에서 자율적으로 수행할 수 있습니다.
- **물리적 샌드박스 격리 (Isolation)**: 로컬 에이전트 스크립트 레벨에서 하드코딩된 물리적 샌드박스가 강제됩니다. 모든 쓰기/수정/삭제 작업은 유저가 지정한 '작업 디렉토리' 내부로만 엄격하게 격리되어, 100% 안전한 자율 AI 제어 환경을 보장합니다.

## 🛠 시작하기

### 사전 요구 사항
- [Node.js](https://nodejs.org/) (v18 이상 권장)
- [PostgreSQL](https://www.postgresql.org/) 또는 SQLite 환경 (`prisma/dev.db`)

### 로컬 환경 설정 안내

1. **레포지토리 클론**
   ```bash
   git clone git@github.com:Rejard/alopop.git
   cd alopop
   ```

2. **의존성 설치**
   ```bash
   npm install
   ```

3. **환경 변수 구성 및 데이터베이스 초기화**
   프로젝트 루트에 `.env.local` 파일을 생성하고 데이터베이스 URL을 설정합니다.
   ```bash
   npx prisma generate
   npx prisma db push
   ```

4. **애플리케이션 실행**
   실시간 소켓 통신을 지원하기 위해 이 프로젝트는 커스텀 express 서버(`server.js`)로 래핑되어 있습니다.
   ```bash
   npm run dev
   ```
   브라우저에서 `http://localhost:3099` 로 접속하여 팩트체크 메신저를 경험해 보세요!
